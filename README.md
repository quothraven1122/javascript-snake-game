## Snake Game
### 배경    
예전에 항해 코육대 게임 만들기 대회를 했었는데    
당시에는 개발 실력이 좋지 않아서 단순한 행맨 게임을 만들었었다.    
개발 실력도 키웠으니 조금 더 복잡한 게임을 만들어 보고 싶다.    
     
목적은 다음과 같다:    
- React에 TS를 쓰는 방법을 배워서 TS를 사용해보고 싶다.
- websocket을 써보고 싶다.

### 개요
두 사람이 랜덤으로 매칭되어 뱀을 조종한다.    
뱀은 맵 밖으로 나가거나, 머리가 상대의 뱀에 부딪히면 죽는다.    
3판 2선제이고 뱀 크기는 8칸이다.
